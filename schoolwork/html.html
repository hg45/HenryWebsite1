What is HTML?

HTML stands for HyperText Markup Language. It is used to make webpages.

Tags

The commands in HTML are called tags. Most tags have opening and closing tags.
For example:

  • <html>your html stuff goes here</html>
  • <body>The stuff that appears on your page goes here</body>

Attributes

Many HTML tags can have attributes. Attributes modify some of the qualities and values associated to the tag. For example:

  • <a href='../index.html'>The href attribute modifies where the link leads to.</a>
  • <img src='../sources/images/ohshc-welcome.gif' title='Welcome~!'>The title attribute shows text when you hover over something.</img>
